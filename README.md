Sistema de Aluguel de Veículos – SQLite Studio
(Projeto Acadêmico)

Este projeto implementa a criação e manipulação de dados para um sistema de aluguel de veículos.  
Foram criadas tabelas, inseridos dados e executadas consultas SQL conforme requisitos acadêmicos.


Arquivos incluídos

01_create_tables.sql
Criação completa das tabelas do banco de dados:
- cliente
- categoria
- veiculo
- reserva
- contrato
- funcionario
- inspecao
- multa
- pagamento

02_insert_data.sql
Inserção de 3 registros por tabela.

03_select_queries.sql
Consultas utilizando:
- SELECT
- WHERE
- JOIN
- ORDER BY
- LIMIT (opcional)
- Filtros em relacionamentos

04_update_delete.sql
Contém:
- 3 comandos UPDATE
- 3 comandos DELETE  
todos com condições claras.


Como executar:

1. Abra seu banco de dados
- SQLiteStudio
2. Execute primeiro o arquivo
- 01_create_tables.sql
3. Em seguida rode
- 02_insert_data.sql
4. Teste as consultas com o arquivo
- 03_select_queries.sql
5. Por último, execute os comandos do
- 04_update_delete.sql

Tecnologias utilizadas

- SQL (DML e DDL)
- SQLite
- GitHub para versionamento
